About

We have developed a system which takes input from an existing dataset path, or scrapes images from WikiArt and saves them, then it passes the image resizer filter and then there are two tests which are optional for this website, which are the Glitcher Test and the watermark test, and then it sends the images to the one of the Generative Adversarial Network model (among three: Vanilla GAN, DC-GAN and W-GAN-gp) or it sends two of the images from different datasets to the Neural Style Transfer Module the final generated image from both of these modules get sent to the Super Resolution Generative Adversarial Network whose job is to sharpen up the image and then finally the images are viewed on this webpage

This Website was basically Our Final Year Project.
